Specs técnicas

Cómo se implementa

Componente vanilla HTML/CSS/JS. Se inyecta en cualquier página GitHub Pages o AWS QAS. Lee `git log` del repo. Crea commit nuevo al restaurar · NO destruye historia.

Trigger

Aplica a

Repos SmartConnection-CodeHub/* con GitHub Pages + ambientes AWS QAS. NO PROD.

Visibility

Env flag

QAS visible · PROD oculto (env=production hide). Garantiza estabilidad cliente final.

Data source

git log real

Lee commits del repo · timestamps · autor · mensaje. NO mock data · datos vivos.

Rollback

Commit nuevo

Restaurar crea commit nuevo con contenido vN. Historia preserved. Re-deploy QAS auto.

Skill Cerebro

/versions-rollback

Cerebro + Pipeline ejecutan rollback. Logged: "Rollback v11→v9 disparado · razón opcional".
