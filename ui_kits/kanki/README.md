# Kanki Street — Mobile storefront

4-screen mobile flow for the Kanki Street streetwear shop running on Jumpseller.

| # | Screen | Anchor element |
|---|--------|----------------|
| 01 | Home | **Curtain hero** — full-bleed navy mesh + editorial display H1 + drop-eyebrow chip |
| 02 | Drops | horizontal drop history with sold-out states |
| 03 | PDP | square hero + serif title + size matrix + heart side-CTA |
| 04 | Cart | line items + summary card + Webpay/MercadoPago footer |

**Brand twist.** The accent colour on Kanki is its own crimson `#D9483A` (= `--danger` token re-purposed as the Kanki brand). Navy + cream + crimson — gold and accent-green stay home for this product.
